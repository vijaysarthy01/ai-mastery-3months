## About This Repo

This is a complete **12-week, project-based learning system** for experienced engineers who want to master agentic AI development. Each week includes:

- What to **watch** (curated free courses from DeepLearning.AI, Anthropic, LangChain)
- What to **build** (a real Python project)
- A **quiz** to test your understanding before moving on

**Stack:** Python 3.11 · Anthropic Claude API · LangGraph · MCP (Model Context Protocol) · CrewAI · ChromaDB

## Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/YOUR_USERNAME/ai-mastery-3months.git
cd ai-mastery-3months

# 2. Create a Python virtual environment
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set up your API key
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
# Get your key at: https://console.anthropic.com/

# 5. Start Week 1
cd week-01
cat README.md
```
